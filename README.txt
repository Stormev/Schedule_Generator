Инструкция для работы:
Программа создаёт рассписание на неделю.

Чтобы сгенерировать расписание требуется -
1. Заполнить все поля в левом браузере, все его листы.
2. Сохраните заполненые поля. Файл -> Сохранить модель файла
3. Нажмите сгенерировать расписание.
Если все было правильно, тогда правые таблицы будут заполнены. (При повторном нажатии, будет другой набор)
4. Экспортируйте в exel. Файл -> Экспортировать в .xlsx

Чтобы отключить субботу зайдите в настройки -> Дополнительный функционал
В найстроках можно указать количество рядов для левого браузера
#-
О программе

Программа была создана для генерации образовательного расписания на 6-дневную неделю.
Позволит быстро и эффективно работать с расписанием.
Есть возможность сохранять и загружать модель + экспорт в exel

Автор программы: Скрипко Виталий
Является 8-м по счёту личным проектом
